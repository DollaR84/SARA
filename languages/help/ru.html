Справочное руководство.

Данное руководство предназначено ознакомить Вас с основными возможностями программы.
SARA (Speak Auto Recognition Agent) - является голосовым помошником, распознающим голосовые комманды, чем упрощает пользование компьютером людям с проблемами со зрением либо с его отсутствием. Данная программа не избавляет пользователя от необходимости некоторых знаний и навыков пользования компьютером, но должна упростить пользование, предоставляя некоторые функции управления для голосовой активации.
Распознавание речи происходит с помощью технологии cmuSphinx.
Также данный помошник имеет возможности напоминания про дни рождения или какие-либо события, важные для пользователя.
В помошнике реализован гибкий механизм многоязычности, благодаря чему он может поддерживать большое количество языков как для произношения, так и для распознавания.



Зависимости помошника

Для работы SARA необходим синтезатор голоса Windows SAPI5.
Некоторые функции доступны только при наличии интернета.
Также часть функций помошника будет работать только в Windows 8 или новее.



Перечень функций SARA.

Символ "|" означает "или", то есть возможность выбора нескольких вариантов произношения комманды.

Для активации произнесите: "окей сара".
