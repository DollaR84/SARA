SARA
Speech Auto Recognition Agent
Все команды доступны для голосового управления
Распознавание происходит благодаря технологии cmuSphinx.
Поддерживаемые возможности:

Версия 1.4:
сервис погоды работает сразу, без необходимости регистрации на специальном сайте
переделана система хранения и использования ресурсов программы

Версия 1.3:
добавлена поддержка программы экранного доступа NVDA
в настройки добавлен выбор речи: NVDA или SAPI

Версия 1.2:
Читает новости из пользовательских rss подписок
добавлена графическая форма настроек
исправлены некоторые баги в работе резервного копирования баз

Версия 1.1:
получает и озвучивает информацию о прогнозе погоды на текущий день и 3 следующих дня
   (для работы данной возможности необходимо зарегестрироваться на сайте "wunderground.com" и получить бесплатный ключ, и указать его в пользовательских настройках)
добавлена графическая форма для создания заметок
добавлена графическая форма для добавления дней рождений
добавлена графическая форма для добавления событий

Версия 1.0:
говорит текущие время и дату
открывает окно компьютера
открывает панель выполнить
открывает charms панель
открывает панель настроек
открывает share панель
открывает панель устройства
открывает панель поиска приложений, файлов или настроек
открывает настройки нескольких мониторов
открывает меню дополнительных настроек программ METRO интерфейса
открывает меню системных утилит
открывает окно специальных возможностей
сворачивает все окна
включает блокировку компьютера
позволяет вести заметки и хранить их в собственной базе
ведет базу дней рождений и уведомляет заранее о приближающемся празднике
добавление записей в базу происходит из файла
   (на стадии разработки находится добавление записей с графической формы)
ведет базу событий и уведомляет заранее о приближающемся событии
добавление записей в базу происходит из файла
   (на стадии разработки находится добавление записей с графической формы)
позволяет создавать резервную копию базы данных помошника и/или любых пользовательских данных
позволяет восстановить из резервной копии базу данных помошника и/или любые пользовательские данные
   (пользовательские данные для резервного копирования необходимо указать в пользовательских настройках)
на стадии разработки находится графическая форма настроек
